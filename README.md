# Campus Hub

A full-stack application with a Next.js frontend and an Express/MongoDB backend.

## Prerequisites
- Node.js (v18+)
- MongoDB

## Setup & Running

### Server (Backend)
1. Navigate to the server directory:
   ```bash
   cd server
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example` and set your MongoDB URI.
4. Start the development server:
   ```bash
   npm run dev
   ```

### Client (Frontend)
1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
